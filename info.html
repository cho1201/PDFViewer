프로젝트 소개

PDF Viewer의 주요 기능과 목적을 소개합니다.

환경과 장소에 구애받지 않는 PDF Viewer

본 프로그램은 어느 환경, 어느 장소에서든 본인의 구글 드라이브 속 PDF를 열람하고
페이지별 메모를 클라우드에 저장하며 관리할 수 있는 개인 맞춤형 웹 애플리케이션입니다.

주요 기능

Google Cloud API 연동

Connect Google Cloud Console에서 클라이언트와 API 발급 및 설정
초기 화면에서 발급받은 API key와 클라이언트 ID로 로그인
Setting /auth/drive.file : 구글 드라이브 파일에 한해 확인·수정·생성·삭제 권한 (메모 저장을 위한 JSON 파일 생성 및 수정)
/auth/drive.readonly : 구글 드라이브 파일 보기 및 다운로드 권한 (PDF 파일 열람)

PDF Viewer 기능

Folder 로그인한 구글 드라이브의 viewer 폴더 속 PDF들을 모두 열람 가능
Note PC(우측 사이드바) 및 모바일(하단 패널)에서 페이지별 메모 작성 가능
Cloud DB 메모 내용은 사용자의 Drive 내 viewer_data 폴더에 .json 파일로 자동 저장되어 기기 간 동기화
UI/UX PC와 모바일 해상도에 자동으로 최적화되는 반응형 레이아웃 제공
